This End-point allows you to search for specific terms in any public channel and get the results. You can use this end-point to : 
- To verify if a particular topic is mentioned , or even 
- To see what people are saying about your brand or company! 
- To categorized a specific post my hashtag 

## How to request

**BASE URL** : ` https://fanatgchannelscraper.herokuapp.com/api/v2` \
**Method** : `GET` \
**End-point** : `/search/:username ` \
**Header** : `token: xxxxxx-xxxxxx-xxxxxx-xxxxxx`

### Params
^
| Name	|Type	| Description | Example | Required     |
| ----  | ----  | ----        | ----    | ----    |  
| search | string | the term you want to search | #sport |  True  |
| after | number | If you want to get the newr posts, pass the starting post id | 234 | False  |
| before | number | If you want to get the older posts, pass the starting post id  | 454 | False  |

### Example



> Dont forget to replace your `token`

#### Search for `shoes`

> ```curl
> curl --location --request GET 'https://fanatgchannelscraper.herokuapp.com/api/v2/search/ethio_market_place?search=shoes' \
> --header 'token: xxxxxx-xxxxxx-xxxxxx-xxxxxx'
> ```
    
#### Search for `shoes` After `1930`

> ```curl
> curl --location --request GET 'https://fanatgchannelscraper.herokuapp.com/api/v2/search/ethio_market_place?search=shoes&after=1930' \
> --header 'token: xxxxxx-xxxxxx-xxxxxx-xxxxxx'
> ```

#### Search for `shoes` Before `1930`

> ```curl
> curl --location --request GET 'https://fanatgchannelscraper.herokuapp.com/api/v2/search/ethio_market_place?search=shoes&before=1930' \
> --header 'token: xxxxxx-xxxxxx-xxxxxx-xxxxxx'
> ```

> | ---- | ---- | ----------- |

> | 400 | Can't find city name xxxxxxx in Ethiopia | Your city is not found in our data set. Please try a different city. |
