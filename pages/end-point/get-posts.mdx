# Get Posts

This end-point allows you to get posts data from any specific telegram channels. You can use this end-point to :
- To get your channel Update
- To get the latest posts from a News, Ecommerce or Job  Channel


## How to request

**BASE URL** : ` https://telegram-scrapper-v2.onrender.com/api/v2` \
**Method** : `GET` \
**End-point** : `/get_posts/:username ` \
**Header** : `token: xxxxxx-xxxxxx-xxxxxx-xxxxxx`

### Params
^
| Name	|Type	| Description | Example | Required     |
| ----  | ----  | ----        | ----    | ----    |  
| after | number | If you want to get the newr posts, pass the starting post id | 234 | False  |
| before | number | If you want to get the older posts, pass the starting post id  | 454 | False  |



### Example

> Dont forget to replace your `token`

#### Get Post from `ethio_market_place`

> ```curl
> curl --location --request GET 'https://telegram-scrapper-v2.onrender.com/api/v2/get_posts/ethio_market_place?search=shoes' \
> --header 'token: xxxxxx-xxxxxx-xxxxxx-xxxxxx'
> ```
    
#### Get Post from `ethio_market_place` After `1930`

> ```curl
> curl --location --request GET 'https://telegram-scrapper-v2.onrender.com/api/v2/get_posts/ethio_market_place?search=shoes&after=1930' \
> --header 'token: xxxxxx-xxxxxx-xxxxxx-xxxxxx'
> ```

#### Get Post from `ethio_market_place` Before `1930`

> ```curl
> curl --location --request GET 'https://telegram-scrapper-v2.onrender.com/api/v2/get_posts/ethio_market_place?search=shoes&before=1930' \
> --header 'token: xxxxxx-xxxxxx-xxxxxx-xxxxxx'
> ```

### Response

```json
[
  {
    "id": 1937,
    "text": "Post Text ...",
    "img": "https://..."
  }
]
```

### Error

> | Code | Messages                                              | Description                                                   |
> | ---- | ----                                                  | -----------                                                   |
> | 400  | NO_VALID_CHANNEL_USERNAME_FOUND                       | Invalid channel username. Please read the documentation.      |
> | 400  | ONLY_CHANNELS_ALLOWED                                 | only channels can be scraped                                  |
> | 401  | NO_POST_FOUND                                         | channels With Empty Post                                      |
> | 401  | NO_VALID_ID_FOUND                                     | post Id is not Valid                                          |
> | 401  | EXPECT_VALID_CHANNEL_USER_NAMES, AS Query PARAMETER   | expect valid multiple channel usernames                       |
> | 401  | MULTIPLE_CHANNEL_USER_NAME_EXPECTED                   | channel usernames have to more than 1                         |
> | 401  | CONNECTION_ERROR_FOUND                                | connection problem occured                                    |
> | 401  | NO_QUERY_FOUND_TO_SEARCH                              | search query not found                                        |
