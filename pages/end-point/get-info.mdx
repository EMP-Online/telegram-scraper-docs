# Get-Info 

This End-point allows you to get informtion about any specific public channel on telegram. 

## How to request

**BASE URL** : ` https://fanatgchannelscraper.herokuapp.com/api/v2` \
**Method** : `GET` \
**End-point** : `/get_info/:username` \
**Header** : `token: xxxxxx-xxxxxx-xxxxxx-xxxxxx`



### Example

> Dont forget to replace your `token`

#### Get info about `Ethio_Market_Place`

> ```curl
> curl --location --request GET 'https://fanatgchannelscraper.herokuapp.com/api/v2/get_info/ethio_market_place' \
> --header 'token: xxxxxx-xxxxxx-xxxxxx-xxxxxx'
> ```

### Response

```json
{
    "profile": "https://...",
    "title": "...",
    "username": "@...",
    "subscribers": "...",
    "photos": "...",
    "links": "..."
}
```


### Error

> | Code | Messages                                              | Description                                                   |
> | ---- | ----                                                  | -----------                                                   |
> | 400  | NO_VALID_CHANNEL_USERNAME_FOUND                       | Invalid channel username. Please read the documentation.      |
> | 400  | ONLY_CHANNELS_ALLOWED                                 | only channels can be scraped                                  |
> | 401  | NO_POST_FOUND                                         | channels With Empty Post                                      |
> | 401  | NO_VALID_ID_FOUND                                     | post Id is not Valid                                          |
> | 401  | EXPECT_VALID_CHANNEL_USER_NAMES, AS Query PARAMETER   | expect valid multiple channel usernames                       |
> | 401  | MULTIPLE_CHANNEL_USER_NAME_EXPECTED                   | channel usernames have to more than 1                         |
> | 401  | CONNECTION_ERROR_FOUND                                | connection problem occured                                    |
> | 401  | NO_QUERY_FOUND_TO_SEARCH                              | search query not found                                        |
